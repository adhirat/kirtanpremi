# Kirtan Premi – Devotional Singer Website

A beautiful, responsive website showcasing the music, biography, events, and philosophy of **Kirtan Premi Prabhu**. Built with **Vite**, vanilla HTML/CSS/JS, and a modern project structure.

## ✨ Features

- Multi‑page static site with smooth navigation
- Responsive layout with immersive hero section
- Dynamic events list (loaded via JS)
- Accessible design and dark‑mode ready

## 📂 Project Structure

```
kirtanpremi/
├─ src/
│  ├─ pages/        # All HTML pages (index.html, bio.html, …)
│  ├─ styles/       # CSS stylesheet (style.css)
│  ├─ scripts/      # JavaScript files (script.js, gallery-data.js)
│  └─ assets/       # Images, icons, audio, etc.
├─ vite.config.ts   # Vite configuration for multi‑page build
├─ README.md
└─ package.json
```

## 🚀 Run Locally

**Prerequisites:** Node.js (>=18)

```bash
npm install          # install dependencies
npm run dev          # start the development server at http://localhost:3000
```

Set your Gemini API key (if needed) in `.env.local`:

```bash
GEMINI_API_KEY=your-key-here
```

## 📦 Build for Production

```bash
npm run build        # generates the `dist/` folder
```

Deploy the `dist/` folder to any static‑hosting service (Firebase Hosting, Netlify, Vercel, etc.).
